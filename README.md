# Simple SSH Shell Client

A lightweight Python SSH client built with Paramiko that allows remote command execution.

## Author
This script is created by **Kofu**.

## Features
- Connect to a remote server via SSH.
- Execute shell commands remotely.
- Interactive prompt showing current user, hostname, and working directory.
- Graceful handling of network errors and authentication failures.
